## ✅ Correções Recomendadas

### Opção 1: Usar Windows Certificate Store (Recomendado)

Python 3.13+ usa automaticamente o Windows cert store. Para Python 3.12, usar wincertstore:

```bash
pip install wincertstore
```

Depois, criar wrapper no orquestrador.py:

```python
import wincertstore
import certifi as certifi_lib

# Usar certificados do Windows
ca_bundle = wincertstore.where()
client = Anthropic(api_key=api_key, http_client=httpx.Client(verify=ca_bundle))
```

---

### Opção 2: Exportar Certificado Raiz do Avast

1. **Obter certificado do Avast:**
```powershell
# Abrir mmc.exe (Microsoft Management Console)
# Adicionar "Certificates (Local Computer)"
# Navegar: Certificates > Trusted Root Certification Authorities
# Procurar por "Avast Web/Mail Shield Root"
# Exportar como DER (.cer)
```

2. **Adicionar ao Python:**
```bash
# Converter para PEM se necessário
openssl x509 -inform DER -in avast_root.cer -out avast_root.pem

# Anexar ao bundle do certifi
cat avast_root.pem >> C:\Users\ANDERSON\AppData\Local\Programs\Python\Python312\Lib\site-packages\certifi\cacert.pem
```

---

### Opção 3: Desabilitar SSL Scanning do Avast (Temporário)

⚠️ **Apenas para teste, não para produção**

1. Abrir Avast
2. Menu → Settings → Protection → Web Shield
3. Desabilitar "Scan encrypted connections (HTTPS)"
4. Reiniciar navegador/terminal
5. Testar API

---

### Opção 4: Usar Certificado Auto-Assinado (Último Recurso)

```python
import ssl
import httpx
from anthropic import Anthropic

# Criar contexto que confia no certificado do Avast
context = ssl.create_default_context()
context.check_hostname = False
context.verify_mode = ssl.CERT_NONE

client = Anthropic(
    api_key=api_key,
    http_client=httpx.Client(verify=False)  # ⚠️ INSEGURO!
```
